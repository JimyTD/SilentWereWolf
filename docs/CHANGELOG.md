# 详细更新日志

## v0.1.0 (2026-03-03)

### 新增
- 游客模式：自动生成用户ID，支持自定义昵称
- 房间系统：创建房间、通过房间号加入、房主管理
- 游戏配置：支持多种预设人数（6/9/12人局），房主可自定义角色数量
- 角色系统：狼人、村民、预言家、女巫、守卫
- 夜晚阶段：各角色按顺序行动（狼人刀人、预言家查验、女巫用药、守卫守护）
- 白天阶段：全体投票放逐
- 胜负判定：狼人全灭好人胜 / 好人阵营人数不足狼人胜
- 断线重连：刷新页面自动恢复房间和游戏状态
- 全局错误提示：操作失败时顶部显示提示信息
- 重复用户检测：同一用户无法同时加入多个游戏中的房间

### 技术
- Docker Compose 部署（nginx 反代 + Node.js 应用）
- 腾讯云 Lighthouse 服务器，端口 8080
- 前后端 TypeScript 类型共享
- Socket.IO 实时双向通信
